Pine Script — אינדיקטור התראות

העתק את הקוד הזה ל-TradingView (Pine Editor). הוא מזהה סיגנלים לפי סטאק האינדיקטורים שלך ושולח התראות עם נתוני JSON. הדבק, שמור, הוסף לגרף, ואז הגדר Alert עם הודעת ה-JSON מהלשונית הבאה.

PINE SCRIPT v5
//@version=5 indicator("Colmex Algo Signal — Nostro Stack", overlay=true) // ═══ Inputs ═══ emaFast = input.int(9, "EMA Fast (Champion proxy)") emaSlow = input.int(21, "EMA Slow (Champion proxy)") rangeLen = input.int(20, "Range Breaker lookback") useVWAP = input.bool(true, "Require VWAP confluence") atrMult = input.float(1.5, "ATR stop multiplier") rrTarget = input.float(2.0, "Risk:Reward target") // ═══ Indicators ═══ emaF = ta.ema(close, emaFast) emaS = ta.ema(close, emaSlow) vwapVal = ta.vwap(hlc3) atr = ta.atr(14) // Champion Trend proxy: EMA cross + slope trendUp = emaF > emaS and emaF > emaF[1] trendDown = emaF < emaS and emaF < emaF[1] // Range Breaker: break of N-bar high/low hh = ta.highest(high, rangeLen)[1] ll = ta.lowest(low, rangeLen)[1] breakUp = close > hh breakDown = close < ll // VWAP confluence vwapOK_long = not useVWAP or close > vwapVal vwapOK_short = not useVWAP or close < vwapVal // ═══ Signals (confluence) ═══ longSignal = trendUp and breakUp and vwapOK_long shortSignal = trendDown and breakDown and vwapOK_short // ═══ Stop & Target ═══ longStop = close - atr * atrMult longTarget = close + atr * atrMult * rrTarget shortStop = close + atr * atrMult shortTarget = close - atr * atrMult * rrTarget // ═══ Plot ═══ plot(emaF, "EMA Fast", color.new(color.aqua, 0)) plot(emaS, "EMA Slow", color.new(color.orange, 0)) plot(vwapVal, "VWAP", color.new(color.yellow, 0), style=plot.style_circles) plotshape(longSignal, "LONG", shape.triangleup, location.belowbar, color.new(color.green,0), size=size.small) plotshape(shortSignal, "SHORT", shape.triangledown, location.abovebar, color.new(color.red,0), size=size.small) // ═══ Alerts ═══ if longSignal alert('{"action":"buy","symbol":"' + syminfo.ticker + '","price":' + str.tostring(close) + ',"stop":' + str.tostring(longStop) + ',"target":' + str.tostring(longTarget) + '}', alert.freq_once_per_bar_close) if shortSignal alert('{"action":"sell","symbol":"' + syminfo.ticker + '","price":' + str.tostring(close) + ',"stop":' + str.tostring(shortStop) + ',"target":' + str.tostring(shortTarget) + '}', alert.freq_once_per_bar_close)
הערה: ה-Pine Script משתמש בקירובים (EMA cross כפרוקסי ל-Champion Trend, range break ל-Range Breaker). זה לא העתק מדויק של אינדיקטורי WealthCharts — אבל לוכד את אותה לוגיקת confluence. אפשר לכוונן את הפרמטרים לפי הביצועים שלך.

תבניות JSON ל-Alerts

כשמגדירים Alert ב-TradingView, מדביקים את ה-JSON הזה בשדה ההודעה. WebhookTrade קורא אותו ומעביר ל-MT4. שים לב: ה-Pine Script כבר מייצר JSON אוטומטית, אבל אלה התבניות אם אתה רוצה לבנות Alert ידני.

JSON — LONG
{ "action": "buy", "symbol": "{{ticker}}", "price": {{close}}, "stop": {{plot_0}}, "target": {{plot_1}}, "lot": 0.1, "comment": "Nostro-Long" }
JSON — SHORT
{ "action": "sell", "symbol": "{{ticker}}", "price": {{close}}, "stop": {{plot_0}}, "target": {{plot_1}}, "lot": 0.1, "comment": "Nostro-Short" }
JSON — CLOSE ALL
{ "action": "close", "symbol": "{{ticker}}", "comment": "Nostro-CloseAll" }

התקנה — שלב אחר שלב

המסלול המלא מ-TradingView עד חשבון קולמקס פרו דרך MT4. עקוב לפי הסדר. בדוק הכל בחשבון דמו לפני שעוברים לאמיתי.

1
הורד MT4 מקולמקס
היכנס לאתר קולמקס פרו והורד את פלטפורמת MetaTrader 4. התחבר עם פרטי החשבון שלך (התחל בחשבון דמו).
2
הירשם ל-WebhookTrade והתקן EA
צור חשבון ב-WebhookTrade. הורד את ה-Expert Advisor שלהם והעתק אותו לתיקיית MQL4/Experts ב-MT4. הפעל אותו על הגרף, ואשר Allow WebRequest בהגדרות.
3
מנוי TradingView Pro
Webhook alerts דורשים מנוי TradingView Pro ומעלה. ללא זה אי אפשר לשלוח התראות אוטומטיות לשרת.
4
הוסף את ה-Pine Script
פתח Pine Editor ב-TradingView, הדבק את הקוד מלשונית "Pine Script", שמור והוסף לגרף. הגדר Alert: בשדה ה-Webhook URL הכנס את הכתובת מ-WebhookTrade, ובשדה ההודעה את ה-JSON.
5
הגדר כללי סיכון
מלא את כללי הסיכון בלשונית "כללי סיכון". התחל שמרני: סטופ יומי $62, מקסימום 20% הון לפוזיציה, R:R מינימלי 1:2.
6
בדוק בדמו — לפחות שבוע
הרץ את כל המערכת על חשבון דמו של קולמקס לכמה ימים. ודא שהסיגנלים מגיעים, ש-MT4 מבצע נכון, ושכללי הסיכון עובדים. רק אחרי שהכל יציב — עבור לאמיתי.
אזהרה כנה: מסחר אלגוריתמי אוטומטי הוא סיכון גבוה. באג בקוד, נתק רשת, או שוק תנודתי יכולים לגרום להפסדים מהירים. המערכת הזו בנויה עם אישור ידני בדשבורד בדיוק בשביל זה — אל תדלג עליו. זה לא ייעוץ פיננסי; האחריות עליך.
